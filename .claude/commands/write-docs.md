Generate clear, developer-friendly documentation.

## Documentation Types

### Program Documentation
- Instruction reference (accounts, params, access control)
- Account structures (fields, sizes, PDA seeds)
- Error codes with descriptions
- Events with field descriptions

### SDK Documentation
- Installation and setup
- Quick start examples
- API reference for each function
- Preset comparison table

### CLI Documentation
- Command reference with examples
- Configuration options
- Common workflows

### Architecture Documentation
- System overview diagram
- Program interaction map
- PDA relationship diagram
- Data flow for key operations

## Standards

- Working code examples that compile
- Tables for structured data
- No AI slop (obvious comments, excessive explanation)
- Link to source files
- Current program IDs and versions
